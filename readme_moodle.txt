PHP FineDiff
==================================
Required PHP composer to install the package
```shell
composer require d4h/finediff
```

Alternative, you can download from https://github.com/D4H/php-finediff/archive/refs/tags/1.0.1.zip
Extract the zip to vendor/d4h/finediff

For more details visit https://github.com/D4H/php-finediff
