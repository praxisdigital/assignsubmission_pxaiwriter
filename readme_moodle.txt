php-htmldiff
==================================
Required PHP composer to install the package
```shell
composer require caxy/php-htmldiff
```

Alternative, you can download from https://github.com/caxy/php-htmldiff/archive/refs/tags/v0.1.14.zip
Extract the zip to vendor/caxy/php-htmldiff

For more details visit https://github.com/caxy/php-htmldiff
